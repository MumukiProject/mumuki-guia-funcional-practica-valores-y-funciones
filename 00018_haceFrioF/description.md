Definí, **usando composición**, la función `haceFrioF`, que indica si una temperatura expresada en grados Fahrenheit es fría. Decimos que hace frío si la temperatura es menor a 8 grados Celsius. 

Ejemplo:

```haskell
ム haceFrioF 50
False
```
